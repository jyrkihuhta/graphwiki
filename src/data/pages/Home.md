## Features

- **Wiki Links** - Connect pages with `[[PageName]]` syntax
- **MetaTable Queries** - Query metadata across pages
- **Graph Visualization** - See your knowledge graph at `/graph`
- **Backlinks** - Every page shows what links to it
- **Tags** - Organize with frontmatter tags
- **Live Preview** - Toggle preview while editing

See the [[Markdown Guide]] for the full syntax reference.
